# Simple Calculator

This Python script allows you to perform basic mathematical operations:
- Addition
- Subtraction
- Multiplication
- Division

## Features
- User-friendly terminal interface.
- Ability to perform multiple calculations until the user chooses to exit.
- Handles division by zero.
